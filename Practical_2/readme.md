# Conversion.c
Converts decimal number to its binary form.
### How to compile:
Command : gcc Conversion.c -o Conversion -lm
### How to run:
Command : ./Conversion


# Sum.c
Calculates the sum of 1/i for i = 0 to 1000 and i = 1000 to 0.
### How to compile:
Command : gcc Sum.c -o Sum
### How to run:
Command : ./Sum
